*Task* *Git commands* --------------------------------------------------------------------------------------------------------------------------------------------------------0----------------------------- Create a new local directory: | git init Connect your local repository to a remote server: | git remote add origin Copy a remote repository to your local system: | git clone Add a specific file to staging (Git) or after a new file is created (Hg): | git add  Add all changes to staging (Git) or all new files (Hg): | git add --all OR git add . Commit changes locally: | git commit -m '' Push changes to your remote repository: | git push  List the status of the files you've changed and those you still need to add or commit: | git status Create a new branch (Git) or bookmark (Hg) and switch to it: | git checkout -b Switch from one branch or bookmark to another: | git checkout List all the branches/bookmarks in your repo with an indication of the one you are on: | git branch Delete the feature branch/bookmark: | git branch -d Push the branch/bookmark to your remote repository: | git push origin Fetch and merge changes on the remote server to your working directory: | git pull Merge two different revisions into one: | git merge Show all changes made since the last commit: | git diff Reset head (hard) if you messed up big time | git reset --hard HEAD~1 <~1 (1 commit back) Reset head (soft) if you messed up little time | git reset --soft HEAD~1 <~1 (1 commit back)
